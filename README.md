## 🧠 Overview

The **AI-Powered Business Growth Platform** is a full-stack, SaaS-style analytics system designed for small and medium businesses.

It transforms raw sales data into:

- 📊 Real-time KPIs  
- 📈 Revenue forecasts  
- 🤖 AI-driven insights  
- 🔍 Explainable ML outputs  
- 📄 Executive reports  

---

## ⚡ Key Features

### 📊 Smart Dashboard
- Revenue, orders, AOV tracking  
- Product & category performance  
- Growth detection & trend analysis  

### 📈 Forecasting Engine
- Prophet-based time series forecasting  
- Linear regression fallback  
- Future revenue prediction (3+ months)  

### 🤖 AI Business Assistant
- Natural language queries  
- Business recommendations  
- Executive summaries using OpenAI  

### 🔍 SHAP Explainability
- Feature importance analysis  
- Model transparency  
- Understand **why sales change**  

### 📄 Executive Reporting
- CSV export  
- PDF report generation  
- AI-generated summaries  

### 🔐 SaaS System
- Secure login system  
- Role-based access control  
- Plan-based feature gating  
- User-level data isolation

## Getting Started

### 1. Clone Repository

git clone https://github.com/noorsaba5/sales-intelligence-dashboard.git
cd sales-intelligence-dashboard
### 2. Install Dependencies
pip install -r requirements.txt
### 3. Run App
streamlit run app.py

## 🔐 Environment Setup (Secrets)

OPENAI_API_KEY = "your_api_key_here"

### Example user (for testing only)
[[USERS]]
username = "demo_user"
password = "your_secure_password"
role = "customer"
plan = "starter"

## 🔒 Security Notes

- Secrets are stored using Streamlit secrets  
- User data is isolated per account  
- Sensitive files are excluded via `.gitignore`  
- Production version should use hashed passwords and a database
